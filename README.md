# cumtb-vpn-script
This is a UserScript that allows you to redirect certain URLs to a VPN server at "vpn.cumtb.edu.cn" on port 8118, in order to access academic sites such as sciencedirect.com, cnki.net, springer.com, nature.com, ieee.org, science.org, wiley.com, tandfonline.com, and wanfangdata.com.cn.

## Installation
1. Install a UserScript manager such as Tampermonkey for Chrome or Greasemonkey for Firefox.
2. Click on the [link](https://raw.githubusercontent.com/Ranhuiryan/cumtb-vpn-script/main/script.js) to install the script.
3. The script will automatically run on any of the specified websites

## Usage
1. Navigate to one of the specified websites.
2. The script will automatically redirect you to the VPN server and you will be able to access the academic content.
